# Associate OS / Signal OS

Static venture-workflow demos built by Isaac Luo.

Public site: https://isaac0314.github.io/Associate-OS/

- `/associate-os/` — founder calls into evidence-linked commitments and drafts
- `/growth-scout/` — European growth watchlist
- `/founder-radar/` — early founder-intent signals
- `/breakout-radar/` — portfolio operating-signal review
- `/deck/` — interview framing deck

The public site is deliberately static. Live X, press, and Granola connectors run only
through localhost sidecars so API credentials and private meeting data never reach
GitHub Pages. Interview datasets are labeled, and drafts always require human review.
